Welcome to Kapua! Eclipse Kapua™ is a modular platform providing the services required to manage IoT gateways and smart edge devices. Kapua provides a core integration framework and an initial set of core IoT services including a device registry, device management services, messaging services, data management, and application enablement. To install Kapua™ on your local machine please refer to the project quick start guide.
